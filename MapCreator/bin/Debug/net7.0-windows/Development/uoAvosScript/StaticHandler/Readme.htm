ORIGINAL ADDITIONS

- Command "ExportStatics" Designed to take all the statics including checking some of the statics to make sure it was a static before exporting. Scripted by DKnight

- Using the power of XML, MapCreator uses these XML files to import your exported statics to create on compiling your map, so everytime you wanted to fix something on you map without altering any of your statics you could do so.


RAVENAL'S ADDITIONS

[x] Command "AreaExportStatics" Designed to target a bounding box area to export into a xml file.

[x] You can now do [AreaExportStatics <Filename> or [ExportStatics <Filename>

For Example:
Using the command: [AreaExportStatics DessertTown-Dock this will create a file in the certain directory which is editable on top of the StaticHandler.cs script.

Refer to the following example:

private const string ExportPath = @"C:\MapCreator\Export\";

When Exporting at anytime, there will be a warning gump to warn you if you wish to do this, so you can't come back to us and say we didn't warn you.


POSSIBLE FUTURE ADDITIONS

[x] ImportStatics, will import all the XML files inside your ExportPath and such, this will have a warning gump before you import

[x] ImportStatics <Filename>, will import a specific filename you requested, that is within your ExportPath.


DIRECTIONS:
How to Export your Statics and use it in MapCreator.

  1. Check your ExportPath to make sure it is the directory you wish to target. If it doesn't exist it will automatically create it, make sure you fix this when you first get this script.
  2. Use either the [ExportStatics or [AreaExportStatics command with <filename> or not to your choice.
  3. When exporting into the xml file, there will be a warning gump, make sure you say okay or cancel if your unsure about this.
  4. When the 3 steps above are complete, goto your ExportPath and grab those files, I would just "Copy" those files, and then place them into your MapCreator Project Folder,

    For Example of what I have is,

C:\MapCreator\MyProject\Map#\Import\

  1. Always make sure in your project folder you have the folder name called: Import. If it doesn't exist right click on your mouse and create a new folder naming it "Import".
  2. After that, do the following steps on how to create your map it then should import the XML code within the Import folder, and add it to the statics#.mul. It will also do the Staidx#.mul files too
